# Numb Crypto Trading World

A hub page that links to 4 trading strategies.

## Folder structure
```
/
├── index.html                    # The hub landing page
└── strategies/
    ├── liquidity-scanner.html    # Strategy 01
    ├── range-pa.html             # Strategy 02
    ├── smc-confluence.html       # Strategy 03
    └── breakout-scanner.html     # Strategy 04
```

## Deploy (GitHub Pages)
1. Create a new GitHub repo. For a clean root URL use the name `abdulbasit-mtn.github.io`.
2. Upload all files keeping the folder structure above.
3. Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / `(root)` → Save.
4. Live in ~1 minute at `https://abdulbasit-mtn.github.io/`.

## Local preview
Just open `index.html` in any browser. All links use relative paths.
